HairBook Setup Wizard

DB: SQLite / MySQL

Kroky: test → vytvořit tabulky → migrace → wipe

1) Připojení k DB a uložení .env

2) Vytvořit tabulky

Spustí api/init-db.php podle nastavení z .env.

3) Migrace dat (JSON z localStorage)

4) Vymazat všechna data

Volá reset-db.php a smaže obsah všech tabulek. Vyžaduje potvrzení a (pokud je nastaven) WIZARD_TOKEN.

5) Kompletní záloha / obnova (všechny tabulky)

Export/Import přes backup.php. Vyžaduje token, pokud je nastaven WIZARD_TOKEN.
